## Installation

### From the Marketplace

Open VS Code Quick Open (`Cmd+P`) and run:

```
ext install publisher.extension-name
```

Or search for the extension in the Extensions view (`Cmd+Shift+X`).

### Install from VSIX

1. Download the `.vsix` file from the
   [releases page](https://github.com/...).
2. In VS Code, run **Extensions → Install from VSIX**.
3. Select the downloaded file.
